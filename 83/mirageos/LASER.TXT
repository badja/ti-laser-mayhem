About
-----
Laser Mayhem is a puzzle game based on Laserstrike for Windows by Kevin Ng.
Version 1.1 is now for MirageOS.


New Features
------------
Laser Mayhem now uses external levels and comes with a level editor for
Windows 3.1 or above. Upon starting the game, you are taken to the level you
were up to when you last played that particular level set.


Installing
----------
Send  LASER.8xp and LASERLEV.8xp file to your calculator. 

If you want to play external levels, you can remove the Standard Levels if you
like by deleting LASERLEV from your calculator. Make sure you send the
external levels after having sent the Laser Mayhem program file.

In-game Keys
------------
Arrows  Move cursor
   2nd  Fire tank (when cursor is on tank)
        Move tank (when cursor is on arrow)
   +/-  Change level
   DEL  Restart level
 CLEAR  Exit game


Level Editor
------------
The level editor runs under Windows 3.1 or above. Just run the LASERED.EXE
file. You need the following files either in the same directory or in the
Windows system directory:

Included in the ZIP file:
    CMDIALOG.VBX
    COMMDLG.DLL
    PICCLIP.VBX
NOT included in the ZIP file:
    VBRUN300.DLL

If you get an error saying that you need VBRUN300.DLL, you can download
it from the Laser Mayhem web page at http://move.to/badja

Once you have saved your level set, all you need to do is assemble it with
ION. You can get ION from the www.ticalc.org or joewing.calc.org

To assemble a level set called "custom.z80", you would type:

    asm custom

Then send the 83P or 8xp file to your calculator (after having sent Laser Mayhem).
